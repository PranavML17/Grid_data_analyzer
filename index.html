⚠ V2G / Interconnection Queue Arbitrage — Not modelled here
V2G aggregation economics and interconnection queue position depend on utility-specific contractual terms, aggregator agreements, and DISCOM pilot program conditions — none of which are in public KPTCL/BESCOM feeds.

Building a calculator on those assumptions would imply false precision. The output would look rigorous but the inputs would be fabricated.

What to actually do: Engage directly with BESCOM/POSOCO for V2G pilot terms. POSOCO is currently piloting V2G through registered Ancillary Service Providers (ASPs). The CEA Grid Code 2023 defines the technical requirements. Start there.

DISCOM connection — BESCOM (Karnataka)
5-step process. Verified June 2026. bescom.karnataka.gov.in
1
Site feasibility
Visit BESCOM sub-division. Request load availability at nearest 11kV feeder for required kVA. Determines if new DT needed.
Timeline: 1–3 days
2
LT-6 / HT-2 application
Apply at bescom.karnataka.gov.in → New Service Connection. LT-6(c) for public DC; HT-2(f) if aggregate >100 kW.
Timeline: 7–15 working days

Demand note + payment
Security deposit (3 months consumption) + service line charges + DT contribution if new transformer needed.
New DT: Rs 4–12 lakh if required
4
CEA electrical safety
CEA Installation Regulations 2006 apply. Licensed wiring supervisor + electrical test certificate before energisation.
